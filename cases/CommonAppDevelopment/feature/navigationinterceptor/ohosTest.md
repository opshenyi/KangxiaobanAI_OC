# Navigation路由拦截案例测试用例

## 用例表

| 测试功能                                                             | 预置条件                    | 输入                                     | 预期输出                                       | 是否自动 | 测试结果 |
|------------------------------------------------------------------|-------------------------|----------------------------------------|--------------------------------------------|------|------|
| 跳转半模态登录页面                                                        | 	点击进入【Navigation路由拦截案例】 | 	进入案例                                  | 从底部弹出半模态登录弹窗，在弹出的半模态登陆页面里，勾选"阅读并同意协议"，点击按钮"手机号码一键登录"，跳转"我的页面"。      | 是    | Pass |
| 在已登录的状态，返回首页，再次点击案例，无需重复登录，可直接进入"我的页面"。 | 	在"我的页面"，并已登录过，是登陆状态    | 在"我的页面"点击返回上级页面按钮，返回上级页面。重新进入案例。      | 此时可直接进入"我的页面"，不需要重复登录。 | 否    | Pass |
| 在"我的页面"退出登陆后，二次登录需重新验证登录。    | 	已退出登录                  | 	在进入"我的页面"状态，点击"退出登录"按钮，退出案例，重新点击进入案例。 | 此时重新进入案例，需要重新登录。  | 否    | Pass |
