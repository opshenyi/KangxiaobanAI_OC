# 自定义日历选择器案例测试用例

## 用例表

| 测试功能    | 预置条件                                        | 输入      | 预期输出                                    | 是否自动 | 测试结果 |
|---------|---------------------------------------------|---------|-----------------------------------------|------|------|
| 日历面板显示  | 	1.启动【自定义日历选择器案例】 | 	点击日期时间 | 正常显示日历面板，显示近两个月的日期，默认当天高亮显示             | 是    | Pass |
| 日历可上下滑动 | 	1.启动【自定义日历选择器案例】 | 	上下滑动日历 | 可上下滑动日历面板，无卡顿                           | 是    | Pass |
| 点击日期可选择 | 	1.启动【自定义日历选择器案例】 | 	点击日期   | 点击日期后，日期面板消失，主页日期变更为选择的日期。当天以前的日期无法点击选中 | 是    | Pass |
