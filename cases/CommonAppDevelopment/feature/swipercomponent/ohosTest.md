# 多层级轮播图案例测试用例

## 用例表

| 测试功能      | 预置条件           | 输入           | 预期输出                   | 是否自动 | 测试结果 |

| 导航点切换     | 	 启动【多层级轮播图案例】 | 	手动滑动图片或自动轮播 | index导航点指示器变为红色，形状变为椭圆 | 否    | Pass |
