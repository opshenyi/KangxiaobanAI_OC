# 列表编辑案例测试用例

## 用例表

| 测试功能   | 预置条件          | 输入                                 | 预期输出                                                             | 是否自动 | 测试结果 |
|--------|---------------|------------------------------------|------------------------------------------------------------------|------|------|
| 新增代办活动 | 	1.启动【列表编辑案例】 | 	1.点击新增按钮<br/>2.选择任一项后点击确认按钮       | 1.弹出活动选择弹窗，弹框中包含滚动选择器<br/>2.弹框关闭，下方列表上方新增刚刚选择的任务                 | 是    | Pass |
| 勾选完成活动 | 	1.启动【列表编辑案例】 | 	点击任务签的选择框                         | 该任务归纳到已完成列表中，且列表名称增加一条删除线                                        | 是    | Pass |
| 编辑代办活动 | 	1.启动【列表编辑案例】 | 	1.点击任务右侧编辑按钮<br/>2.输入文字后点击右侧对号    | 1.任务文字可编辑，自动获焦，弹出键盘，右侧编辑按钮变为√，左侧选择框小时<br/>2.任务面板恢复至初始状态，任务名称改动成功 | 是    | Pass |
| 删除代办活动 | 	1.启动【列表编辑案例】 | 	1.左滑未完成活动，点击删除<br/>2.左滑已完成活动，点击删除 | 1.未完成列表减少一项<br/>2.已完成列表减少一项                                      | 是    | Pass |
